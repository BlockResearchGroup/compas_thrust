compas_tno.algorithms

initialise_problem

Initialise the problem for a given Form-Diagram and return the set of matrices and vectors to optimise.

initialise_form

Initialise the problem for a Form-Diagram and return the FormDiagram with independent edges assigned

z_from_form

Relaxation of Form-Diagram.

zlq_from_qid

Calculate z’s from independent edges.

zlq_from_q

Calculate z’s from all q’s.

q_from_qid

Calculate q’s from all qid’s.

z_update

Built-in update of the heights in the Form-Diagram (only z-coordinates can change).

reactions

Compute and plot the reaction on the supports.

set_up_nonlinear_optimisation

Set up a nonlinear optimisation problem.

set_up_convex_optimisation

Set up a nonlinear optimisation problem.

run_optimisation_scipy

Run nonlinear optimisation problem with scipy

run_optimisation_MATLAB

Run convex optimisation problem with MATLAB.

run_optimisation_MMA

Run convex optimisation problem with MMA.

find_independents

Find inndependent edges in a given equilibrium matrix.

independents_exclude

Find inndependent edges with certain to exclude.

independents_include

Find inndependent edges with certain to include.

inds_incl_excl

Find inndependent edges with certain to exclude and include.

check_independents

Run checks to verify the independennts.

check_horizontal

Run checks to verify of a certain horizontal distribution can be taken by the fixed form diagram.
