compas_tno.diagrams

Diagrams

FormDiagram()

The FormDiagram class imports the attributes and set ups from compas_tna.diagrams.FormDiagram and include some functionalities useful for the assessment of masonry structures.

ForceDiagram()

Mesh-based data structure for force diagrams in TNO.

Rectangular diagrams

create_cross_form(cls[, xy_span, ...])

Helper to construct a FormDiagram based on with an orthogonal arrangement and main diagonal.

create_cross_diagonal(cls[, xy_span, ...])

Helper to construct a FormDiagram based on cross discretiastion including diagonals that go along the quads.

create_cross_with_diagonal(cls[, xy_span, ...])

Helper to construct a FormDiagram based on cross discretiastion with orthogonal arrangement and diagonals applied to all corners.

create_fan_form(cls[, xy_span, ...])

Helper to construct a FormDiagram based on fan discretiastion with straight lines to the corners.

create_ortho_form(cls[, xy_span, ...])

Helper to construct a FormDiagram based on cross discretiastion with orthogonal arrangement and diagonal.

Circular diagrams

create_circular_radial_form(cls[, center, ...])

Helper to construct a circular radial FormDiagram with hoops not equally spaced in plan.

create_circular_radial_spaced_form(cls[, ...])

Helper to construct a circular radial FormDiagram with hoops not equally spaced in plan, but equally spaced with regards to the projection on a hemisphere.

create_circular_spiral_form(cls[, center, ...])

Helper to create a spiral Form-diagram and set common attributes.

Linear diagrams

create_arch_form_diagram(cls[, H, L, x0, ...])

Helper to create a arch linear form-diagram with equaly spaced (in 3D) nodes.

create_linear_form_diagram(cls[, L, x0, ...])

Helper to create a arch linear form-diagram with equaly spaced (in 2D) nodes.
